# AI Resume Screening System

## Overview

AI Resume Screening System is a full-stack web application that analyzes resumes against job descriptions and provides intelligent matching insights.

## Features

* Upload Resume (PDF)
* Extract Resume Text
* Skill Detection
* Job Description Matching
* Match Score Calculation
* Missing Skills Identification
* AI Suggestions
* Progress Bar Visualization
* Recruiter Dashboard Statistics
* Analytics Charts
* Analysis History
* PDF Report Generation
* MySQL Database Integration

## Tech Stack

### Frontend

* React.js
* Axios
* Chart.js
* jsPDF

### Backend

* Node.js
* Express.js
* Multer
* pdf-parse

### Database

* MySQL (XAMPP)

## Installation

### Backend

```bash
cd backend
npm install
node server.js
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

## Project Workflow

1. Upload Resume PDF
2. Extract Resume Content
3. Analyze Skills
4. Compare with Job Description
5. Generate Match Score
6. Detect Missing Skills
7. Provide Suggestions
8. Save Analysis in MySQL
9. Display Analytics Dashboard
10. Download Analysis Report
